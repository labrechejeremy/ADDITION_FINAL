Pool de minage — comment faire

Tu peux miner en solo ou créer/rejoindre un pool. Voici le flux recommandé pour un setup propre et rapide.

1) Préparer le node

  • Démarrer le node en mode mainnet.
  • Vérifier que la hauteur de chaîne augmente.
  • Activer RPC sécurisé (token RPC obligatoire).

2) Choisir le mode

  • Solo : toutes les rewards pour toi, variance plus élevée.
  • Pool privé : partage hashrate entre tes machines.
  • Pool public : payouts réguliers selon contribution.

3) Connecter les mineurs

  • Configurer l’adresse wallet de réception.
  • Pointer les workers vers l’endpoint pool.
  • Surveiller rejet shares, latence et uptime.

4) Exploitation continue

  • Mettre alertes sur hashrate et peers.
  • Sauvegarder `data/` régulièrement.
  • Garder les clés/tokens uniquement côté serveur.
Conseils pool:\n • Faible hashrate global → préférer pool pour lisser les rewards\n • Hashrate élevé + infra stable → solo possible\n • Toujours séparer wallet froid (trésorerie) et wallet chaud (opérations)
