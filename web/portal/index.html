Quick Start (Node + Backend + Portal)

  1. Build node: cmake -S . -B build ; cmake --build build
  2. Start node: build/additiond.exe
  3. Start portal backend: python web/portal/addition_portal_backend.py
  4. Open this page and verify Node: ONLINE
  5. Use Swap Console and Token Studio below
Tip: If your node runs on another machine/port, update API base URL in web/portal/js/app.js.
